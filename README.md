# motion_planner
==============

motion planner in javascript version

This is an implementation of path planning algorithm.

## Algorithm

- NF1 - potential fields 
- BFS - Breath first search

---

# How to run

1. You have to install `node` and `yarn` in advance
2. Install package
    ```bash
    yarn
    ```
3. Run up at local
    ```bash
    yarn dev
    ```
